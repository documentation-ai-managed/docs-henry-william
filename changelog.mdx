<Update id="v1.2.0" label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added full-text search across all documentation pages with fuzzy matching and filters.
- Introduced version history for pages, allowing you to revert changes easily.
- New template system for consistent documentation layouts.

## Improvements

- Optimized page load times by `>40%` through lazy loading.
- Enhanced mobile responsiveness for better editing on tablets.

## Bug Fixes

- Fixed sync issues in multi-user editing sessions.
- Resolved export failures for large documentation spaces.

</Update>

<Update id="v1.1.0" label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Real-time collaboration with live cursors and comments.
- Integrated GitHub sync for version-controlled documentation.
- Customizable sidebars with drag-and-drop organization.

## Bug Fixes

- Corrected authentication token refresh logic.
- Fixed table rendering in preview mode.

````jsx
// Example: New GitHub sync usage
import { syncDocs } from '@henry-william/docs-sdk';

await syncDocs({
  repo: 'your-org/your-docs',
  branch: 'main',
  apiKey: 'YOUR_API_KEY'
});
````

</Update>
