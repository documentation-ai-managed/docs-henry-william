## Document Creation and Editing

Create new documents instantly. You edit using familiar markdown syntax enhanced with interactive components like tabs and code groups.

<Steps>
  <Step title="Create Document" icon="plus">
    Click the `New Document` button in your workspace.
  </Step>
  <Step title="Edit Content" icon="edit">
    Use markdown for text, headings, and lists. Add components for richer layouts.
  </Step>
  <Step title="Preview & Publish" icon="eye">
    Preview changes live, then publish to make available.
  </Step>
</Steps>

For example, embed code snippets directly:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  async function fetchDocs() {
    const response = await fetch('https://api.example.com/docs');
    return response.json();
  }
  ```
  ```python
  import requests

  def fetch_docs():
      response = requests.get('https://api.example.com/docs')
      return response.json()
  ```
</CodeGroup>

## Organization and Folder Structures

Organize docs into folders and subfolders. You drag and drop to rearrange, creating intuitive hierarchies for large projects.

```
Workspace Root
├── Guides/
│   ├── Quickstart.mdx
│   └── Authentication.mdx
├── API/
│   └── Reference.mdx
└── Changelog.mdx
```

This structure keeps related docs together, making navigation seamless.

## Search and Filtering Capabilities

Search across all documents with full-text capabilities. Filter by tags, folders, or dates.

<Tabs>
  <Tab title="Basic Search" icon="search">
    Type keywords in the global search bar to find matches instantly.
  </Tab>
  <Tab title="Advanced Filters" icon="filter">
    Narrow results:

    | Filter Type | Example |
    |-------------|---------|
    | Folder     | `folder:guides` |
    | Tag        | `tag:api` |
    | Date       | `after:2024-01-01` |
  </Tab>
</Tabs>

## Collaboration and Sharing Options

Invite team members and share docs securely. You control permissions per document or folder.

<Expandable title="Permission Levels" default-open="true">
  - **View**: Read-only access
  - **Edit**: Modify content
  - **Admin**: Manage permissions and folders
</Expandable>

Generate shareable links with expiration dates. For integrations, use webhooks:

```javascript
// Webhook example for doc updates
fetch('https://your-webhook-url.com/webhook', {
  method: 'POST',
  headers: { 'Authorization': 'Bearer YOUR_TOKEN' },
  body: JSON.stringify({ event: 'doc.updated', docId: '123' })
});
```

<Callout kind="info">
  Real-time collaboration shows edits live, like Google Docs.
</Callout>
