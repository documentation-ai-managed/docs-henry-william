## Overview

Henry William provides a centralized space to organize, manage, and publish your project documentation. You create structured docs using MDX with powerful components, ensuring your team stays informed and productive. Access everything from a single dashboard at `https://dashboard.henrywilliam.com`.

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://henrywilliam.com/register` with your email.
  </Step>
  <Step title="Set Up Space" icon="settings">
    Create a new documentation space and invite your team.
  </Step>
  <Step title="Add Your First Page" icon="file-plus">
    Use the editor to write MDX content. Start with the introduction template.
  </Step>
  <Step title="Publish" icon="globe">
    Preview and publish your docs to share the live site.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Embed docs widget in your app
  import { HenryWilliamDocs } from '@henry-william/docs-sdk';
  
  const docs = new HenryWilliamDocs({ spaceId: 'your-space-id' });
  docs.render('#docs-container');
  ```
  ```python
  # Python integration example
  from henry_william import DocsClient
  
  client = DocsClient(api_key='YOUR_API_KEY')
  client.embed(space_id='your-space-id', target_id='docs-container')
  ```
</CodeGroup>

## Brand Guidelines

Maintain consistency across your documentation.

Use the primary brand color `#3B82F6` for interactive elements like links and buttons. Pair it with neutral grays for text (`#1F2937`, `#6B7280`).

```css
:root {
  --brand-primary: #3B82F6;
  --text-primary: #1F2937;
}

.button {
  background-color: var(--brand-primary);
  color: white;
}
```

<Callout kind="info">
  Download the full brand kit from the dashboard under Settings > Brand.
</Callout>
